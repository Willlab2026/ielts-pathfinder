词根前缀基础 (Basic Roots) - Part 1

在英语中，超过 60% 的词汇来源于拉丁语或希腊语。掌握这些词根词缀，就像获得了一把解开单词密码的钥匙。今天我们先来学习最常见的 8 个基础词根。

一、词根 "spect" = 看 / 观察

spect 来自拉丁语 specere，意为"看"。由它构成的词汇往往与观察、观看有关。

inspect [in- = into] → 深入地看 → v. 检查，仔细审视
respect [re- = back] → 回头看（在意）→ v./n. 尊敬，重视
spectacular [spect + -acular 形容词后缀] → 值得看的 → adj. 壮观的，引人注目的

二、词根 "vert/vers" = 转 / 转变

vert 和 vers 都来自拉丁语 vertere，意为"转动"。这个家族的词汇非常多，在雅思阅读和写作中频繁出现。

convert [con- = together] → 一起转过来 → v. 转换，改变（信仰/用途）
diverse [di- = apart] → 分开转向不同方向 → adj. 多样的，不同的
reverse [re- = back] → 转回去 → v./adj. 颠倒，反转

三、记忆技巧

词根记忆的关键是建立"词根-含义"的直接关联，而不是逐个字母背单词。试着用下面的方法：

  • 看到 spect → 直接想到"看"
  • 看到一个新词 retrospect → retro（向后）+ spect（看）→ 回顾
  • 不要翻译成中文再理解
